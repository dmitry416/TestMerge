## Merge - Тестовое задание

## О проекте

**Merge** - игра в жанре "кликер" с элементами прогрессии и инвентаря. Игрок сбрасывает кирки для разрушения блоков, зарабатывает деньги и улучшает свой арсенал.

### Основные особенности:
- **Система инвентаря** с объединением предметов
- **Прогрессия** уровней и улучшений (уровни бесконечные)
- **Кроссплатформенность** - работает на PC и мобильных устройствах
- **Сохранение прогресса**

## Архитектурные решения

### 1. **Система сохранения (Save System)**
```csharp
public interface ISaveSystem
{
    public void SaveData();
    public void LoadData();
}
```
Необходимо сохранять состояние игры между сессиями с возможностью расширения.

**Решение**: 
- **ISaveSystem** - унифицированный интерфейс для всех сохранений
- **Регистрационная модель** - SaveManager автоматически находит и уведомляет все системы
- **Изоляция данных** - каждая система знает как сохранять свои данные

**Преимущества**:
- Легко добавлять новые системы сохранения
- Автоматическое управление жизненным циклом
- Четкое разделение ответственности

### 2. **Синглтон-менеджеры**
```csharp
public class GameManager : MonoBehaviour
{
    public static GameManager Instance { get; private set; }
    public static int CurrentLevel { get; private set; }
}
```
Глобальный доступ к менеджерам без создания жестких связей.

**Решение**: 
- **Lazy initialization** - менеджеры создаются при первом обращении
- **DontDestroyOnLoad** - сохранение состояния между сценами
- **Статические свойства** - удобный доступ из любого места кода

### 3. **Система перетаскивания в инвентаре**
```csharp
public class Cell : MonoBehaviour, IPointerDownHandler, IPointerUpHandler, IPointerEnterHandler
```
Создание интуитивной системы перетаскивания для мобильных устройств.

**Решение**:
- **Input System Package** - современная система ввода
- **Raycast-based detection** - точное определение целевых ячеек

### 4. **Генерация уровней**
```csharp
System.Random random = new System.Random(SeedLvlGeneration + _blocksCount * GameManager.CurrentLevel + i);
```
**Детерминированная генерация** - одинаковые "случайные" уровни для всех игроков.
